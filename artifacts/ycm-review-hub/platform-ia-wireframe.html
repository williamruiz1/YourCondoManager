YourCondoManager — Platform Information Architecture

Before / After nav simplification · 2026-06-30 · audit doc: docs/platform-ia-simplification-audit-2026-06-30.md

The core problem

  • Triple-stacked layers: zone label → redundant hub card-grid page → the actual page (which is already a sidebar sub-item). Every zone shows its children twice.
  • Two-door conflation: Maintenance is a page AND a Work Orders tab; Inspections is a page AND a Vendors tab; recurring dues live under both Chart of Accounts and Assessment Rules; reconciliation under both Reports and an orphan admin route.
  • Buried governance: Meetings / Elections / Compliance are hidden tabs reachable only at 4 clicks deep — none is a sidebar item.
  • Biggest win: delete the hub layer → every zone goes from 3 clicks to 2, zero feature loss.
