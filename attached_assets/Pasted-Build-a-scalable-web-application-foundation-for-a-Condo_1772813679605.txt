Build a scalable web application foundation for a Condo Property Management Platform.

This platform will eventually manage condo associations, governance records, HOA finances, owner registries, documents, and board operations.

For this first build, focus ONLY on creating the technical foundation and the first core module.

Tech Stack

Use the following stack:

Backend: Node.js with Express

Database: PostgreSQL

ORM: Prisma

Frontend: React with Vite

Styling: Tailwind CSS

API: REST API

Authentication: simple session-based authentication

File storage: support document uploads

Structure the project so it can scale into a SaaS platform later.

Code Quality Requirements

Use TypeScript everywhere

Use Prisma migrations

Separate controllers from services

Use environment variables

Use proper error handling

Important Constraints

Do NOT attempt to build:

payment processing

email automation

HOA fee calculations

advanced document AI ingestion

Those will be added later.

Focus on building a clean, extensible architecture.

Deliverables

When finished, the project should include:

running React frontend

running Express API

PostgreSQL database with Prisma

working CRUD for the core entities

ability to upload documents

simple admin dashboard
